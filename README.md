# E-Commerce Sales Analysis with Python & SQL

## 📊 Overview
This project performs exploratory data analysis on an e-commerce dataset to identify sales trends, profitability, and customer behavior. The analysis includes data cleaning, visualization, and SQL-based insights.

## 🛠 Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn)
- SQL (SQLite)
- Jupyter Notebook

## 🔍 Key Features
- Data cleaning and preprocessing (handling missing values, duplicates, date formatting)
- Exploratory Data Analysis (EDA)
- Sales and profit analysis by category, region, and time
- Data visualization (bar charts, line charts, pie charts, scatter plots)
- SQL queries for business insights

## 📈 Key Insights
- Technology category generates the highest sales
- Tables and Bookcases are loss-making sub-categories
- Sales peak during Q4 (Oct–Dec), indicating seasonal trends
- West region contributes the highest overall sales

## ▶️ How to Run
1. Install required libraries:
   pip install pandas matplotlib seaborn

2. Open the Jupyter Notebook:
   sales_analysis.ipynb

3. Run all cells to view analysis and visualizations

## 📁 Dataset
- Superstore Sales Dataset (CSV format)
